# METALCORE - 钢管钢板供应链管理平台

> 鲁西（聊城）供应链管理有限公司官方网站

## 项目概述

METALCORE 是一个专业的钢铁供应链企业门户网站，展示公司在钢管、钢板等主材供应、大宗物资贸易及供应链监管仓库领域的专业服务能力。

### 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite 5
- **路由**: React Router DOM (HashRouter)
- **国际化**: i18next (支持 6 种语言)
- **样式**: CSS + Tailwind CSS
- **图标**: SVG 矢量图标

## 设计理念

### 1. 专业工业美学

网站采用**深色主题 + 橙色强调**的配色方案，体现钢铁行业的专业性和力量感：

```
主色调:
- 深蓝灰背景: #0f172a, #1e293b
- 橙色强调: #f97316 (品牌活力)
- 白色文字: 高对比度，确保可读性
- 渐变效果: 增强视觉层次
```

**设计原则**:
- **克制的装饰**: 避免过度装饰，让内容成为焦点
- **数据驱动展示**: 用数字和事实说话，建立信任
- **工业质感**: 通过几何图形、网格布局体现工业美学
- **写实配图**: AI 生成的高质量工业场景图片，增强真实感

### 2. 信息架构

网站采用**扁平化导航 + 锚点滚动**的信息架构：

```
主导航:
首页 → 产品 → 解决方案 → 关于我们 → 技术实力 → ESG → 新闻 → 联系
```

**层级设计**:
- **一级页面**: 独立内容模块，完整展示信息
- **锚点导航**: 关于我们页面使用锚点滚动，避免深层嵌套
- **CTA 策略**: 每个页面底部设置明确的行动召唤
- **滚动置顶**: 页面切换自动回到顶部

### 3. 多语言国际化

支持 6 种语言，覆盖主要市场区域：

| 语言 | 代码 | 目标市场 |
|------|------|----------|
| 中文 | zh | 国内市场 |
| English | en | 国际市场 |
| Español | es | 拉美市场 |
| Français | fr | 非洲市场 |
| العربية | ar | 中东市场 |
| Русский | ru | 独联体市场 |

**实现方式**:
- i18next 统一管理翻译
- 语言切换即时生效，无需刷新
- RTL 支持（阿拉伯语）
- 所有页面 100% 翻译覆盖

## 视觉风格

### 色彩系统

```css
/* 品牌色 */
--primary: #f97316;        /* 橙色 - 活力、行动 */
--primary-dark: #ea580c;   /* 深橙 - hover 状态 */

/* 背景色 */
--bg-dark: #0f172a;        /* 深蓝灰 - 主背景 */
--bg-darker: #020617;      /* 更深 - hero 区域 */
--bg-card: #1e293b;        /* 卡片背景 */

/* 文字色 */
--text-primary: #ffffff;   /* 主文字 */
--text-secondary: #94a3b8; /* 次要文字 */
--text-muted: #64748b;     /* 辅助文字 */

/* 边框 */
--border: #334155;         /* 分割线 */
```

### 排版系统

```css
/* 字体 */
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;

/* 标题 */
h1: 3rem / 48px, font-weight: 700
h2: 2.25rem / 36px, font-weight: 600
h3: 1.5rem / 24px, font-weight: 600

/* 正文 */
body: 1rem / 16px, line-height: 1.6
small: 0.875rem / 14px
```

### 图标风格

采用**线性 SVG 图标**，统一风格：

- 线宽: `stroke-width: 1.5`
- 圆角端点: `stroke-linecap: round`
- 尺寸: 48×48px（功能图标）、24×24px（导航图标）

**图标类型**:
- 制造、研发、质量、自动化（技术实力）
- 碳中和、水资源、循环、培训（ESG）
- 供应、贸易、仓库（解决方案）

## 动效设计

### 1. 滚动显示动画

使用 IntersectionObserver 实现滚动触发动画：

```css
/* 动画类型 */
.reveal        → 从下向上淡入
.reveal-left   → 从左向右滑入
.reveal-right  → 从右向左滑入
.reveal-scale  → 缩放淡入

/* 时间函数 */
transition: all 0.8s cubic-bezier(0.16, 1, 0.3, 1);
```

### 2. 数字计数动画

统计数据使用计数动画增强视觉冲击：

```
示例:
15+ 年行业深耕
50K+ 月交付吨数
99.8% 合格率
```

### 3. Hero 入场动画

页面加载时的序列动画：

```
序列:
1. 标题淡入 (0.1s 延迟)
2. 副标题淡入 (0.2s 延迟)
3. 按钮淡入 (0.3s 延迟)
4. 背景图片缩放 (持续 20s)
```

## 开发指南

### 安装

```bash
git clone https://github.com/dugd-insight/steel-portal.git
cd steel-portal
npm install
```

### 开发

```bash
npm run dev
# 访问 http://localhost:5173
```


## 维护说明

### 添加新语言

1. 创建 `src/i18n/locales/xx.json`
2. 在 `src/i18n/index.ts` 中导入
3. 在 `LanguageSwitcher.tsx` 中添加选项

### 修改内容

- **翻译文本**: 修改 `src/i18n/locales/*.json`
- **页面结构**: 修改 `src/pages/*.tsx`
- **样式**: 修改 `src/pages/pages.css` 或组件内联样式
- **动画**: 修改 `src/styles/animations.css`

### 更新部署

```bash
# 标准构建
npm run build

# 带标签构建（推荐）
node scripts/build-with-tag.js

# 将 dist/ 或 dist-tagged/xxx/ 部署到服务器
```
