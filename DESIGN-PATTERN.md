# METALCORE 设计模式参考
> 基于 https://www.creditsteelpipe.com 分析提炼

## 一、配色体系

| 用途 | 颜色 | 代码 |
|------|------|------|
| 主色 | 深蓝 | `#1a365d` |
| 辅色 | 橙色 | `#d97706` |
| CTA强调 | 亮橙 | `#f59e0b` |
| 背景 | 白色 | `#ffffff` |
| 次级背景 | 浅灰 | `#f5f5f5` |
| 文字主色 | 深灰 | `#333333` |
| 文字次色 | 中灰 | `#666666` |

## 二、布局模式

### 1. 导航栏：固定顶部，Logo左 + 导航中 + 联系右
### 2. Hero：全宽轮播图 + 白色文字覆盖（阴影可读）
### 3. 信任条：深色背景 + 电话/邮箱/WhatsApp图标
### 4. 核心优势：4列等分（图标+大数字+描述）
### 5. 关于我们：左图右文（50/50）
### 6. 产品展示：分类标签 + 4列网格卡片
### 7. 项目案例：大图 + 标题 + 描述
### 8. 页脚：4列（公司/导航/产品/联系）

## 三、信息层级
- 产品分类名 → 具体产品名 → 产品图片 → 规格参数
- 命名规范：标准号+材质+产品类型+用途

## 四、配图风格
- Hero：工厂全景/生产线（真实场景）
- 产品：白底或浅灰底，产品居中
- 项目：施工现场/应用场景
- 图标：简洁线性，蓝色或灰色

## 五、核心原则
1. 白底黑字，高对比度
2. 信息优先，无多余装饰
3. 模块化布局，层次分明
4. 信任导向（资质/经验/客户数）
5. 转化导向（联系方式醒目）
